（271-280）饭饭AniTalks

发表于 2019-09-06 | 分类于 动漫观影手记

271、进击的巨人 Season3 Part.2

剧情很刺激，引发对自由的解释，诠释了生命的意义。
还是那句话，建议都看看。

272、拾又之国

国人漫画走出去的一小步，为了在内地流通，尺度略小，但是不失为一部剧情流畅的冒险叙事动画，虽然最后的反派战斗有点智商丢失，但是无伤大雅。

273、川柳少女

对，就是这个笑容。
这部番，香菜配音，值一观。
七七子，喜用川柳，不擅谈。
川柳者，一行十字，三四三。
男主角，面目狰狞，实心善。
看本剧，满满狗粮，坐不安。
唯所憾，追番之后，难人言。

274、滿腦都是○○的我沒辦法談戀愛

H向，尺度比最强插边球出包还是差了点，但是纯情啊。啊！纯情？

275、為什麼老師會在這裡！？

这个严格意义上来说算里番了，但是这剧情，不是老司机根本开不动这样的车，这里就推荐一下嗶咔上面可以看漫画，其他不多说。

276、文豪野犬

只看了第一季，只能说剧情创新，通篇流畅没毛病，但是不是我喜欢的类型。

277、网络胜利组

虐狗纯爱番，异世界但不打怪，每集都会玩游戏但主线剧情在现实世界，网充和现充的不得不说的故事，故事差不多04年国内网文的水平，做成动画还是很精彩的。

278、犬屋敷

大佬的漫画作品不给你整这些有的没的，有啥东西我都能画，反倒是人物都比较狰狞？不知道是不是我的错觉。
改编成动画这个优势就没那么明显了，虽然剧情略扯，但是看完还是让人挺感动的。
警告：本篇含有大量流血、暴力场景，小孩子就别看了。

279、高分少女

超级好看，加上OVA目前一共15集，期待结局。
虽然小春作为女二注定失败，但是还是很同情她，明明是我……额……还真不是我先来。
感觉漫画结尾男主女主两人也不会走在一起的样子，估计是留个悬念，什么家庭落差太大要好好努力配得上女主之类的才行然后就结束这样，难受。

280、鬼滅之刃

还没看完，总体还是非常不错的，剧情精彩，打斗刺激，而且是搞笑风格的。不错。
