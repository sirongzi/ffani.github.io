（101-110）记下我看过的那些日本动漫

发表于 2018-03-15 | 分类于 日本のアニメの記録記事

101、监狱学园

节奏感非常强，而又不失幽默。 一部让人脑洞大开的作品，每个人都刻画的很有特点，还有理事长每句话最后一字停顿重读简直停不下来……

102、狼与香辛料&狼与香辛料Ⅱ

俺可是贤狼赫萝！
就这样狼和罗伦斯的旅途开始了，很喜欢这种中世纪背景风格，旅行商人什么的，吃苹果的狼什么的，基本每一季都是两个故事组成的，慢热型的高潮，故事标题都是狼と啥啥啥，看着很悲剧的标题其实是很温情的哈，没有打斗没有魔法，只有旅行商人的小小心机和一人一狼的温情满满。
两季都很好看，不过第二季明显铺垫不够，不过还是很棒哈，小说故事最后貌似是终成眷属啦，然后生一堆猴子，哦，是狼……人什么的……

103、神的记事本

很棒的情感类剧情番……虽然尼特侦探什么的，但是我个人感觉此番的推理和侦探的部分其实都可以忽略的，大家看看这个脸，这个去掉头发和一休、千反田爱瑠一模一样的脸……
其实这个番主要是讲了一个道理，每个人都有思想上想不开的地方，文艺点就是说人由欲望而生，孰能看空一切，冥冥中……滚犊子，在说什么哩！这段cut，导演，重新来啊，也就是说人都有自己的脆弱，要找到属于自己的坚强，你需要属于你的朋友，比如尼特什么的。矛盾并不是不能化解，遗憾并不是一定要抱着度过终身，眼泪不接受后悔，但是接受理解和帮助，总有个人比你还要爱你，那就用你的所有去爱她吧。
如爱丽丝所谓，对于那些已经逝去的人们，已经失去的事物，能够有所意义的职业，在这个世界上只有两个：也就是作家和侦探，只有作家，才能让这些在梦中复活重现，只有侦探，才能吧这些从墓中挖出来，并整理为信息。侦探的本质是死者的代言人，将失去的语言从墓穴里挖出，为了守护死者的名誉而伤害生者，为了安慰生者而侮辱死者，所以理应是不受欢迎也不被理解的工作。我稍微觉得：自己也可以这样过日子吧！ 学会像普通人一样欢笑，像普通人一样生气，像普通人一样哭泣。 失去了许多东西，我才终于发现自己其实并非一无所有。
原作小说很多很棒的话语，但是更慢……还是很棒的番吧，后面稍微有点催泪。

104、命运石之门

一开始就有可恶的剧透狗给我说女主不是まゆり，然后……但是比起Christina来说我还是更喜欢まゆり的，特别是她的嘟嘟噜。 冈部伦太郎是个中二，但是也就只有中二才能承受那样大的压力和无数次的时间旅行并且看到まゆり的死了，剧中的其他角色也塑造得很成功，最喜欢菲利斯喵喵了，铺垫很长感觉前半部分大概有8、9集的样子都是在铺垫，说实话前面很无聊……但这还是无法掩盖平行世界和时间穿越带来的冲击感和压抑、快速的有趣节奏，铺垫很成功，后面还是很带感的。 剧情大概就是（剧透啦，剧透啦，剧透啦，重要的事情说三遍！）男主发明时间机器，尝试使用，切换到女二会死的时间轴，想改变这个结果，尝试一切方法想回到最初的时间轴，结果发现这个时间轴上女一会死，最后靠着朋友的女二从未来带来的时间旅行机器穿越回去“改变”女一会死的结果，结局大团圆，世界上没有了时间机器，只有男主对于这三周的记忆。话说其中有一个很有意思的现象，男主在撤销D-mail的时候，说服了很多人，大概就是从易到难的顺序（也是时间倒序），最后发现最难说服的反而是自己。还是很有趣的。 带轮回和时间轴、平行世界什么的都是很受人欢迎的，参照小圆……

105、凉宫春日的忧郁（2009）

SOS团什么的，灰常棒！看了这番感觉自己高中都白费了系列……
还有漫无止境的八月，播了8集，一共重复了15532次。
很温馨很欢乐很热血很青春，说实话一部以爱情为隐线的小说讲的是青春，而且用了最烂的温泉、灯会、社团等等为载体但拍出了令人啧啧的效果，无疑是一部神作，很开心，中二满满但是毫无违和之处。
看了才懂系列，我发现很多系列的后宫番都有明显抄袭这个的痕迹……

106、叛逆的鲁鲁修

鲁鲁修是一部很棒的番，比如十一区什么的，制作背景好像是映射的二战后日本一直处于美国的控制状态下……然后，自己揣摩去吧。
鲁鲁的能力很变态，所以从一开始就很强力，而且性格带一点小小的邪恶，是很能勾起大家的观看欲望的。但这并不是无敌，所以剧情的波澜起伏以及矛盾的层出不穷都是很能勾人心魂的。
剧中有夏莉这样的悲情女主，也有C.C这样的默默辅助，以及红头发的（被欺骗下的）支持；枢木朱雀作为鲁鲁的铁哥们，和鲁鲁也是由分到合，最终战无不胜……鲁鲁最后的结局也是意料之外（于是有了车夫党……）。总之，每个人物的塑造也是十分成功的，个性鲜明，呼之欲出。
男孩子们都应该去看看的番。

107、ToLove-RuTrouble&MottoToLove-RuTrouble&ToLove-RuTroubleDarkness（出包女王一二三季）

最喜欢美柑和梦梦了还有唯和贫乳娜娜！
妹妹美柑的表情简直魔性，给大家两张图感受一下

还有这个霓虹小学生书包——

简直勾人犯罪，讲道理这书包挺贵的，不过能保♂护♀小学生。（有各种颜色哦）
总之，尺度最大的表番动画，据说漫画也是表漫之最……
另，一定要看无修！圣光会忽悠你~

108、一拳超人

非常的诙谐幽默，而且逼格特别高……又是一部装逼高快感神作……
什么都是一圈秒掉，好轻松哦！
怎么做到的啊？！每天跑跑步，做做俯卧撑什么的啊，要坚持哦！
地底人？海王？改造人？外星人？都不够看，一拳秒掉！秒掉！
ONE PUNCH MAN!

109、伊克西翁传说 DT

诙谐搞笑的异界穿越嘴炮拯救世界的故事。
主人公一行的画风很渣，反而是一开始和主人公一行做对的因克古尼特比较“有型”。
设定很扯，大概是个反萌差的思路，但是……最后一点都不萌只剩下了搞笑。也许是因为画风不够出色吧，毕竟冷番……
主角比较废，反而是比较喜欢因克古尼特一行，每个人的性格都塑造得十分鲜明，而且他们在一起很和谐友爱的呢，这就是一个team吧，当然公主和伪娘（后来变成了真娘……）小动物保护协会战士还有我们的嘴炮主角也是性格突出的，但是为什么我还是感觉不萌……
所以哈哈哈的笑就好了，剧情什么的，都是扯淡。

110、夏日大作战

超展开，情节有点学美国电影的感觉——看到开头猜到结局系列。
但是里面蕴含了很多日本人文情怀，比如大家族老人打电话，一个家族的互相帮忙，每个家都有一个支心骨什么的。这里有必要吐槽下台版的老奶奶配音，让我想起了各种韩国泡菜剧……
情节还是很吸引人的，但是我并不是很喜欢这种抽象的战斗画面（小圆即视感……）。
还有里面的那个什么牌，“花札”，还有配音说的“来来”“几光”“猪鹿蝶”什么的……好像很复杂的样子哩。
还有被女主拉回老家的原因简直泥垢，还有一开始的男主的好朋友呢，就出了事过后给了点情报露了个面然后就没啦。不出个重置版重新选另外一个么……
